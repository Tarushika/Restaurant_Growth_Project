# Restaurant Growth Potential Analysis

## 📌 Project Overview

This project was completed as part of the Unified Mentor Data Analyst Internship. The objective was to analyze restaurant performance, identify growth opportunities, and provide business recommendations using data analytics and machine learning techniques.

## 🎯 Objectives

* Analyze restaurant revenue, profitability, and operational performance.
* Segment restaurants into meaningful business groups using clustering.
* Identify restaurants with high growth potential.
* Generate actionable business recommendations.
* Develop an interactive dashboard for data visualization.

## 📂 Dataset

The dataset contains restaurant-related information such as:

* Revenue and profit metrics
* Order volumes
* Delivery channel performance
* Cost and operational metrics
* Growth indicators

## 🛠️ Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Streamlit
* Google Colab

## 🔍 Methodology

1. Data Cleaning and Preprocessing
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Data Standardization
5. PCA (Dimensionality Reduction)
6. K-Means Clustering
7. Growth Potential Score (GPS) Development
8. Dashboard Creation

## 📊 Feature Engineering

The following business metrics were created:

* Total Revenue
* Total Profit
* Cost Pressure
* Channel Leverage
* Growth Momentum

## 🤖 Machine Learning Techniques

### PCA (Principal Component Analysis)

Used to reduce dimensionality and improve clustering performance.

### K-Means Clustering

Restaurants were segmented into four clusters:

* High Revenue Leaders
* Overextended Low Return
* Stable Local Performers
* High Growth Scalable Leaders

## 📈 Growth Potential Score (GPS)

A custom scoring model was developed:

GPS = 0.4 × GrowthFactor + 0.4 × TotalProfit + 0.2 × ChannelLeverage

This score helps identify restaurants with strong future growth potential.

## 📌 Key Findings

* Four distinct restaurant segments were identified.
* High Revenue Leaders generated the strongest financial performance.
* Overextended restaurants showed profitability challenges due to high costs.
* Delivery-focused restaurants demonstrated higher scalability.
* Growth Potential Score effectively highlighted expansion-ready restaurants.

## 💡 Recommendations

* Invest in High Growth Scalable Leaders.
* Optimize operational costs for Overextended restaurants.
* Strengthen delivery channel strategies.
* Use GPS for future investment and expansion decisions.

## 📊 Dashboard

An interactive Streamlit dashboard was developed to:

* Visualize cluster distribution
* Analyze revenue and profit metrics
* Display Growth Potential Scores
* Explore restaurant recommendations
